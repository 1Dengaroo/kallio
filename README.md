# Kallio - The AI Video Editor

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

This project uses Next.js, Tailwind, ShadCN, Remotion, Clerk, Prettier, TypeScript

Learn about [the Remotion Player](https://www.remotion.dev/docs/player/)
